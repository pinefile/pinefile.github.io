Run

Run shell commands and log instead of returning output as shell function does.

Usage#

const { run } = require("@pinefile/pine");
module.exports = {
example: async () => {
await run("tsc");
},
};

Options#

All Execa options can be used except stdin, stdout and stderr that's are defaulting to inherit.

To change them you should use shell function instead and read about the default options for shell function.
